About this Project


This is a personal Project which utilizes Airgov weather api for searching air quality based on the city zip code.

In order to use this app enter the zip code in the Search box and click Lookup. This will fetch the air quality of that area.

AirNow is a US centralized weather data system which provides different quality control parameter for air quality
